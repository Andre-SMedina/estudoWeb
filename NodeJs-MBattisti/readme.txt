* npm init -y ---> cria rápido o package.json sem perguntar
* npm install express nodemon mongoose--> pode instalar vários de uma vez
* npm install --save-dev chalk--> instala mas registra no package.json que é uma dependência só para o desenvolvedor e não para o projeto
* npm update--> atualiza todos os pacotes
* npm update nodemon--> atualiza apenas o pacote nodemon
*npx npm-check-updates -u--> verifica se tem algum pacote para atualizar
* npm link nodemon--> caso esteja no global e não funcione, tem que executar esse comando pra puxar ele do global
